Smart Wallet est un site web qui offre à l’utilisateur une interface claire et structurée pour visualiser ses revenus et ses dépenses.

Le site est composé de deux pages principales :

La page d’accueil (Home), qui contient un tableau de bord (dashboard). Cette page permet à l’utilisateur de consulter l’ensemble de ses revenus et dépenses, ainsi que de visualiser les données sous forme de tableau pour une meilleure lisibilité et organisation.

La page d’ajout de revenu/dépense, qui permet à l’utilisateur de saisir de nouvelles données à l’aide d’un formulaire. Les informations soumises sont ensuite enregistrées dans la base de données.
